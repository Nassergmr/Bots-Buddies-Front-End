Summary
BotsBuddies is a chat app that lets users interact with multiple AI models in one place. Designed for quick testing, comparison, and exploration of different assistants. Powered by GitHub Models and Azure AI.

Features
- Chat with multiple AI models
- Switch models instantly with Next.js routing
- Conversations saved via LocalStorage
- Simple, intuitive chat UI

Tech Stack
Frontend: Next.js, Tailwind CSS
Backend: Node.js, Socket.IO

API Provider
GitHub Models

Deployment
Frontend: Vercel
Backend: Railway
